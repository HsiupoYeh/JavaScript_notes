# JavaScript_notes
JavaScript簡稱JS。

## 運行特色:  
  + Single Thread (單線程)  
  + Synchronous (同步)
    > Javascript 一次只執行一件事，程式會逐行執行 。  
    > 所以普通的程式碼都是同步(阻塞式)的，實現非同步(非阻塞式)的程式碼是依賴運行環境提供的特殊功能。

## 運行環境:
  + 瀏覽器
  + Node.js

## 實際上的狀況
  + 配合運行環境，JS可以撰寫出同步與非同步的程式碼。
  + 非同步的程式碼風格:
    + 舊式的 callback 風格
    + 新式的 promise 風格
